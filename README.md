## Development
```console
pip install poetry
cd swingify2
poetry install
```
### Run
To run cli entrypoint:
```console
poetry run swingify2 --help
```

### Tests
```console
poetry run tox
```

### Docker
To run with docker
```console
docker build -t Swingify .
docker run swingify2:latest swingify2 --help
```
